Git

Git is software tracking changes in any set of files, usually for coordinating work among programmers developing source code during software development. It is a DevOps tool for source code management. It is free and open-source that enables multiple developers working together

Branches

A branch is a version of the repository that diverges from the main working project which is the master branch It allows work to continue without changing the master branch. A Git project can have more than one branch. These branches are a pointer to a snapshot of your changes. When you want to add a new feature or fix a bug, you spawn a new branch to summarize your changes.
